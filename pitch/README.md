# Meguri — Pitch Deck

A self-contained, 5-slide investor/judge pitch deck for **Meguri 巡り**, built to match
the product's visual identity (deep-indigo surface, vermilion + gold palette, Japanese
typography, asanoha pattern).

## Present it
Open `pitch/index.html` in any browser and go fullscreen.

| Action | Keys |
| --- | --- |
| Next / previous | `→` `←` · `Space` · `PageUp/Down` · swipe |
| Jump to slide | number keys `1`–`5`, `Home`, `End` |
| Fullscreen | `F` |
| Export to PDF | `P` (or `Cmd/Ctrl+P` → Save as PDF, **landscape**) |

You can also click the dots / arrows in the bottom-right.

## The five slides
1. **Title** — "Tokyo is hiding something. Let's go find it." (hook + one-liner)
2. **Problem** — record tourism, but everyone sees the same ten spots
3. **Solution** — the core loop: choose a guide → get a quest → walk & scan → collect
4. **Why it works** — retention engine (XP, Kakera, postcards, parties) + the AI stack
5. **Market & the ask** — Tokyo today, every city tomorrow; how it monetizes; the ask

## Notes
- **Fully offline-safe:** all icons are inline SVG and every layout has a CSS fallback,
  so nothing breaks if you present without Wi-Fi.
- Fonts load from Google Fonts when online (they degrade to clean system fonts otherwise),
  and the product screenshots/avatars load when online (a styled CSS placeholder shows if not).
- Market figures on slide 5 are illustrative framing for a pitch — edit the copy directly
  in `index.html` to match your numbers before presenting.
